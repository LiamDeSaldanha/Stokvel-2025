# How to setup project

## With Docker

1. Install Docker + Docker Compose
2. Run (once):
   ```bash
   docker compose build 
   ```
3. Afterwards use:
    ```bash
   docker compose up
    ```

4. Create a env file in root adding your OPENAI_API_KEY=secret
